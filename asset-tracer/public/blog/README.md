# Blog Images Directory

Place blog post featured images in this directory.

## Current Images Needed

- `5-signs-outgrown-spreadsheets.png` - Featured image for "5 Signs Your Business Has Outgrown Spreadsheets for Asset Tracking"

## Image Specifications

- Recommended size: 1200x630 pixels (OpenGraph standard)
- Format: PNG or JPG
- File naming: Use the blog post slug as the filename

## How to Add Images

1. Save your image file in this directory (`asset-tracer/public/blog/`)
2. Use the filename that matches the `image` property in `blog-posts.ts`
3. The image will automatically be displayed on the blog post page
